## Installation

```bash
$ npm install
```

## Running the app

```bash
$ docker-compose up -d mongodb

$ npm run start:dev
```

## Running Docker

```bash
$ docker-compose up
```

## Docs

# Rest

# Graphql

$ http://localhost:3000/graphql
